# Support

Need help with CPKB?

## Questions and Usage Help

Open a GitHub issue and include:

- Your operating system
- How you installed CPKB (`pipx`, `pip`, Homebrew, or source)
- The command you ran
- The output or error message

## Bug Reports

Use the bug report issue template and include reproduction steps. If the bug touches your snippet database, do not upload private database files; share only a minimal example or sanitized export.

## Security Issues

Please do not open public issues for vulnerabilities. Follow the reporting steps in [SECURITY.md](SECURITY.md).
